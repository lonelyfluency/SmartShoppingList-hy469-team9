Before starting the client:
    - install the leap motion SDK (as specified in the relative tutorial slides)
    - connect the leap motion's device USB to your computer
    - start the Leap Motion Control Panel app
    - In the Troubleshooting tab, verify that all the statuses are green, otherwise click on Recalibrate Device
    - In the main.js file, change the FRONTEND_URL and FRONTEND_PORT variables according to your project specifications

Then, in a terminal, execute (IF ON WINDOWS, USE POWERSHELL):
    sudo apt-get update
    npm install 

To run the client:
    npm start
